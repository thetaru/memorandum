# システムの評価指標
## H31S P13
稼働率をR(0<R<1)とか置いて計算する。  
稼働率はこのように考えます。
```
直列
   +---------------+      +---------------+
   |   　稼働率　   |      | 　　稼働率　   |
---+               +------+               +---
   |      0.9      |      |      0.8      |
   +---------------+      +---------------+
 
 稼働率 = 0.9 × 0.8 = 0.72 (72%)

並列
       +---------------+
       |   　稼働率　   |
   +---+               +---+
   |   |      0.9      |   |
   |   +---------------+   |
---+                       +---
   |   +---------------+   |
   |   | 　　稼働率　   |   |
   +---+               +---+
       |      0.8      |
       +---------------+
稼働率 = 1 - ((1 - 0.9) × (1 - 0.8)) = 1 - 0.02 = 0.98 (98%)
※ 両方が同時に停止する確率を求めるのが重要です
```

## H28F P15
MIPSは、CPUが**1秒間に**実行可能な命令回数を**百万単位**で表した指標です。  
例えば、100MIPSの場合、1秒間に実行可能な命令数は10,000万回です。
