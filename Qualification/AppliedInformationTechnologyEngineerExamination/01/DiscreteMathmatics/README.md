# 離散数学
## ■ H29S P1
論理和(OR)、論理積(AND)、排他的論理和(XOR)は結合法則を満たす。  
例えば、排他的論理和の真理値表を見てみると実際に(A XOR B) XOR C = A XOR (B XOR C)となっていることがわかる。

|A|B|C|(A XOR B) XOR C|
|:---|:---|:---|:---|
|0|0|0|0 XOR 0 = 0|
|0|1|0|1 XOR 0 = 1|
|0|1|1|1 XOR 1 = 0|
|0|0|1|0 XOR 1 = 1|
|1|0|0|1 XOR 0 = 1|
|1|1|0|0 XOR 0 = 0|
|1|0|1|1 XOR 1 = 0|
|1|1|1|0 XOR 1 = 1|

|A|B|C|A XOR (B XOR C)|
|:---|:---|:---|:---|
|0|0|0|0 XOR 0 = 0|
|0|1|0|0 XOR 1 = 1|
|0|1|1|1 XOR 1 = 0|
|0|0|1|0 XOR 1 = 1|
|1|0|0|1 XOR 0 = 1|
|1|1|0|1 XOR 1 = 0|
|1|0|1|1 XOR 1 = 0|
|1|1|1|1 XOR 0 = 1|

わからなくなったら、ベン図か真理値表を書けばいいと思う。

## ■ R1F P2
ベン図を描くときは、演算子(論理和、論理積など)を分けて考えるとわかりやすいと思う。
