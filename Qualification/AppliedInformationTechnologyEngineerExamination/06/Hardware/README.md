# ハードウェア
## H28S P24
SoC(System on a Chip)は、1つの半導体チップ上にシステムに必要な一連の機能を集積する回路の設計方法です。  
占有面積の削減、高速化、低消費電力化、コスト削減などのメリットがあります。

## H28S P22
パワーゲーティングは、集積回路内の使用していない演算回路ブロックへの電源を遮断することでリーク電流の削減を図る技術です。
