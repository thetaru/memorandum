# aws-cli
# § INSTALL
```
# curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
# unzip awscliv2.zip
# ./aws/install
```
```
### version確認
# aws --version

```
### config
# aws configure
