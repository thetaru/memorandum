# ログの管理・運用
アプリケーションが出力した各種ログをEKS上で収集、保管、監視、可視化する方法について解説します。
## ■ 4-2-1 アプリケーションの出力場所
k8sでは、アプリケーションのログ出力先を標準出力とするのが推奨です。  
出力されたログは、`kubectl logs <Pod名>`で参照できます。(標準出力なのでアプリケーションが出力した内容)
