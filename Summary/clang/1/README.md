## 1.3.2 Hello World プログラム
次のソースコード`main.c`を書きます。
```c
#include <stdio.h>
int main(void)
{
  printf("Hello World!");
  return 0;
}
```
コンパイルして実行したら画面上に"Hello World!"と表示されたら成功です。
```
$ gcc main.c -o main.o
```

```
$ ./main.o
```
```
Hello, World
```

### 1.3.4 エスケープシークエンス
Cには特殊文字があって以下のようなものがある。

|記号|意味|
|:---|:---|
|\n|改行|

|\\?|疑問符|
