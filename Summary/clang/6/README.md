# ポインタ
## 6.1 ポインタとは
ポインタとは**メモリのアドレスを保存する**変数です。
```
---------------------------------------------------------------------------------
|       |       |  int  |       |       |       |       |  int* |       |       |
|   ~   |       |   5   |       |       |   ~   |       | 0x100 |       |   ~   |
|       |       | 0x100 |       |       |       |       |  150  |       |       |
---------------------------------------------------------------------------------
```
