# Linux
# ■ About Notation
コマンドプロンプトを使用する際は次のように記載します。  
また指定がない場合ユーザは`root`とします。  
なおコメントアウトは`###`始まりとします。
```
### これはコメントです
# echo 'これはコマンドです'
```
