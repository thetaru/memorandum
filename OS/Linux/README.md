# Linux
# ■ About Notation
コマンドプロンプトを使用する際は次のように記載します。  
また`#`はユーザが`root`、`$`はユーザがroot以外であることを意味します。  
なおコメントアウトは`###`始まりとします。
```
### これはコメントです
# echo 'これはrootです'
$ echo 'これもuserです'
```
