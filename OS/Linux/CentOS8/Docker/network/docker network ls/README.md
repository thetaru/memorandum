# ネットワークの一覧表示
## Syntax
```
# docker network ls [option]
```
### e.g.
#### ネットワーク一覧表示
```
# docker network ls
```
```
NETWORK ID          NAME                DRIVER              SCOPE
ade8c08e5cf4        bridge              bridge              local
412e4fa42521        host                host                local
f933759686b5        none                null                local
```
