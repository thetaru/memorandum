# React 開発環境構築
## ■ インストール
### ● Node.js
#### Node.jsとnpmのインストール
```
# yum install nodejs npm
```
#### 最新のNode.jsのインストール
```
# npm install n -g
```
#### 古いNode.jsのアンインストール
```
# n stable
# yum remove nodejs npm
```
#### create-react-appのインストール
```
# npm install create-react-app -g
```
