## Defaultvers (★)
NFSサーバとやり取りする際に初めに使用するプロトコルのバージョンを指定する。  
もしNFSサーバ側でDefaultversで指定したバージョンが対応していない場合は違うプロトコルを使用して再度接続する。  
※ mountの`v`オプションを使うと挙動が見れます
