# Esxi 監視
# 手順
## ディスカバリルール作成
`設定`-`ディスカバリ`より以下のようなディスカバリルールを作成します。
```
名前: Esxi_discovery
IPアドレスの範囲: <EsxiのIPアドレスまたはEsxiのIPアドレスを含むIPアドレスの範囲>
チェック: ICMP Ping
```
